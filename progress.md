用户反馈财报下载失败，且失败原因不清晰

3. 只有一个数据源（东方财富）
4. 错误信息不够详细，用户无法排查

**增强方案**:
1. **重试机制**: HTTP请求添加指数退避重试（最多3次，间隔1s/2s/4s）
2. **超时优化**: 增加超时时间至30秒
3. **备用数据源**: 东方财富失败时自动尝试腾讯财经
4. **错误改进**: 提供更详细的错误信息和故障排查建议
5. **部分成功**: 只要有部分年份数据成功就返回，不完全失败

### Session 2026-04-13 (Part 12) - CSS强制对齐最终方案

**问题**: 
- 四个图表右侧价格轴不对齐
- 底部日期显示不全
- 多次修改lightweight-charts配置无效

**根本原因**: lightweight-charts库会根据价格数字宽度自动调整轴宽度，无法通过配置强制固定

1. 固定图表宽度为800px
2. 使用CSS flex布局，确保四个图表容器完全等宽
3. 每个图表容器设置固定宽度和高度
4. 右轴宽度固定为80px
5. 布林带图表240px高度（含60px时间轴区域）

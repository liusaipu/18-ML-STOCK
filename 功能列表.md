# 18步法自选股票分析系统 — 功能列表

## 一、自选股票管理

| 功能 | 描述 |
|------|------|
| **自选列表** | 支持增删改查，最多 100 只股票 |
| **搜索添加** | 按股票代码、名称或拼音首字母实时搜索，下拉提示前 10 条结果 |
| **拖动排序** | 支持拖拽调整自选顺序，自动持久化到本地 |
| **股票代码库** | 内置 A 股（沪/深/创业板/科创板/北交所）+ 港股，共 8,239 只 |

## 二、财报数据获取

| 功能 | 描述 |
|------|------|
| **网络下载财报** | 对接东方财富 HSF10 API + akshare 港股接口，自动探测 companyType，下载资产负债表、利润表、现金流量表（支持 A 股、港股） |
| **CSV/Excel 导入** | 支持从同花顺手动下载的 CSV 宽容导入，自动识别报表类型和年份 |
| **数据校验** | 下载后自动对接 datacenter-web API 校验关键科目（总资产、负债、权益、货币资金） |
| **历史版本归档** | 每次导入/下载自动保留最近 3 批历史数据，支持查看时间、来源、年份 |
| **当前数据导出** | 将当前生效的财务数据导出到本地 |
| **历史数据导出** | 将指定批次的历史财务数据导出到本地 |

## 三、18 步财务分析引擎

| 功能 | 描述 |
|------|------|
| **18 步分析法** | 完整实现十八步财报分析法，涵盖审计、资产质量、偿债、盈利、现金流、ROE、成长、分红等维度 |
| **Beneish M-Score** | 完整模型计算财报操纵风险 |
| **可比公司分析** | 支持添加 3~5 家可比公司，自动下载财报并计算关键指标均值/最高/最低/排名百分位 |
| **多年度趋势对比** | 当前公司 vs 可比均值的关键指标多年度对比 |
| **百分制评分** | 基于 18 项维度计算综合得分（每项 5.56 分，按偏离程度扣 1~3 分），输出评级与建议 |
| **缓存机制** | 数据未变时提示缓存；数据没变但可比公司变化时支持覆盖上次报告 |
| **强制重新分析** | 数据未变情况下，弹窗提供「强制重新分析」按钮，覆盖生成新报告 |

## 四、计算过程溯源

| 功能 | 描述 |
|------|------|
| **指标溯源** | 17 个计算步骤均生成结构化计算日志（ROE、毛利率、资产负债率、营收增长率、净利润现金含量、存货周转率、总资产周转率、期间费用率、主营利润率、分红率等） |
| **章节触发** | 在报告各关键章节（3.1、3.2、4.1、7.1、9.1、11.1）标题旁显示 ❓ 按钮 |
| **溯源抽屉** | 点击后右侧滑出抽屉，展示公式、原始数据输入、分步计算过程、口径说明 |
| **多指标切换** | 若某章节涉及多个指标，抽屉顶部支持下拉切换 |

## 五、报告生成与管理

| 功能 | 描述 |
|------|------|
| **14 模块专业报告** | 生成标准投资分析框架：执行摘要、换手率深度分析、公司基本面、行业横向对比、十五五政策匹配、实时行情、RIM 估值、技术面分析、ML 预测、智能选股、芒格逆向检查、巴芒清单、舆情监控、综合投资建议、结论与附录 |
| **Markdown 输出** | 报告以 Markdown 格式渲染，支持一键下载为 `.md` 文件 |
| **历史报告管理** | 支持查看最新/历史报告、切换查看、删除报告 |
| **报告内搜索** | 支持关键词搜索报告内容，按回车依次跳转匹配项并高亮 |
| **目录跳转** | 顶层提供「跳转章节」下拉框，快速定位到各模块 |
| **去重精简** | 执行摘要中去除与 3.1 重复的表格；移除报告正文前的目录（由顶部跳转替代） |

## 六、实时行情与技术指标

| 功能 | 描述 |
|------|------|
| **实时行情** | 对接东方财富为主 + 腾讯财经 fallback，15 分钟本地缓存，展示最新价、涨跌幅、换手率、量比、流通市值、PE、PB 等 |
| **K 线图表** | 展示近期 K 线走势（基于 Apache ECharts） |
| **技术面分析** | 日内价格位置、跳空分析、量价关系、短期技术倾向评分 |
| **换手率深度分析** | 换手率分级与流动性评级 |
| **实时估值评分** | 基于 PE/PB 计算估值得分（0~100），输出估值评语（低估/合理/偏高/过高） |

## 七、舆情与政策

| 功能 | 描述 |
|------|------|
| **社交媒体情绪监控** | 对接东财研报 + 东财公告 + 新浪新闻三重 fallback 做情绪分析，输出情绪得分、热度指数、利好/风险关键词、最新舆情摘要 |
| **十五五政策匹配度** | 基于行业输出政策匹配评级、重点方向、解读摘要 |

## 八、前端交互体验

| 功能 | 描述 |
|------|------|
| **深色/浅色主题** | 右上角一键切换，偏好持久化到 localStorage |
| **三栏布局** | 左栏（自选列表）、中栏（股票信息/操作/亮点风险）、右栏（报告展示） |
| **左栏宽度可调** | 支持拖动调整侧边栏宽度 |
| **亮点与风险摘要** | 分析完成后中栏自动显示 ✅ 亮点 / ⚠️ 风险简要摘要 |
| **股票基本资料卡片** | 显示行业、上市日期、实控人/董事长、总市值、PE、PB、更新时间和一键刷新 |
| **概念 & 风口** | 展示 Wind 行业概念与概念标签 |

## 九、技术栈

- **后端**：Go 1.26 + Wails v2
- **前端**：React 18 + TypeScript + Vite
- **图表**：Apache ECharts（K 线、技术指标）、recharts
- **Markdown**：react-markdown + remark-gfm + rehype-slug + rehype-raw
- **搜索/拼音**：pinyin-pro
- **本地存储**：基于文件系统的 JSON + CSV 解析
